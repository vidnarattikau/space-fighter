# NEON SPACE FIGHTER ⚡🛸

纯前端霓虹太空战机，打开即玩，适合和同学炫耀。

```
    /\\   ★  NEON SPACE FIGHTER  ★   /\\
   |==|   🚀 炫酷霓虹 · 粒子爆炸 · 多武器  |==|
    \\/__________________________________\\
```

## 特性亮点
- 🔥 **赛博霓虹风**：深空渐变 + 霓虹发光粒子，屏幕震动 & 爆炸特效。
- 🎮 **丝滑操作**：WASD/方向键 + 空格/鼠标射击；移动端虚拟摇杆 + 射击按钮。
- 🚀 **武器系统**：默认激光 → 散射 → 激光束 → 导弹追踪，拾取道具升级。
- 👾 **敌人梯度**：普通机、精英机、周期 Boss（多段血条弹幕）。
- 🎁 **道具**：护盾、加速、武器升级、炸弹（全屏清屏）。
- 📈 **积分 & 连杀**：连杀加分、最高分本地榜（localStorage）。
- 🔊 **纯代码音效**：Web Audio API 合成射击/爆炸/拾取音效，无需外部音频。
- 🕹️ **60fps 波次制**：波次提示，越打越难，Boss 每 4 波登场。
- 📱 **移动端适配**：触屏虚拟摇杆 + 射击按钮，防误触设计。

## 控制
- **移动**：WASD / 方向键 / 虚拟摇杆（触屏）
- **射击**：空格 / 鼠标点击 / 触屏射击按钮
- **炸弹**：B
- **暂停**：P

## 玩法
- 清理敌机获得分数，连杀叠加额外加分。
- 拾取道具升级武器、加速或补血，炸弹可瞬间清屏。
- 每隔几波出现 Boss，击败可掉落奖励，注意走位和节奏。
